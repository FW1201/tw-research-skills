# 雙菱形各階段對話腳本
# Double Diamond Phase Dialogue Scripts

---

## 開場白（Session Start）

每次啟動時，先顯示雙菱形框架視覺化：

```
╔══════════════════════════════════════════════════════╗
║          學術研究計畫雙菱形引導系統                      ║
║                                                      ║
║  ◇ 探索    ◆ 聚焦    ◇ 發展    ◆ 交付              ║
║ (Discover)(Define) (Develop)(Deliver)                ║
║                                                      ║
║  我們將一步步從模糊想法走到正式計畫書 📄              ║
╚══════════════════════════════════════════════════════╝
```

開場詢問：
「歡迎！我會引導您用設計思考的雙菱形框架發展研究計畫。
請先告訴我：您對哪個研究領域或議題感到好奇，或是有什麼教學/工作上的觀察讓您想深入了解？
（可以很模糊，一、兩個詞也沒關係）」

---

## PHASE 1：Discover 探索 🔍

### 目標
協助使用者「發散」思考，看見研究領域的全貌。

### 開場語
「很好！我們現在進入**探索階段**——就像先把地圖打開，看看整片研究地景。
讓我先去查一下這個領域的最新動態，給您一個全貌。」

### Web Search 策略
```
搜尋1：「[使用者輸入的領域] 研究趨勢 2022 2023 2024」
搜尋2：「[領域] research gap 台灣 OR Taiwan」
搜尋3：「[領域] systematic review recent」
```

### 呈現搜尋結果的格式
搜尋完成後，整理成：

```
🗺️ [領域名稱] 研究地景一覽

📌 熱門研究方向（近5年）：
  1. [方向A] - [一句說明]
  2. [方向B] - [一句說明]
  3. [方向C] - [一句說明]

🔓 尚待填補的研究缺口：
  • [缺口1]
  • [缺口2]

🌏 台灣學術脈絡中的特殊面向：
  • [本地特色]
```

### Phase 1 引導問題（逐一問，等候回答）

**Q1-1**（理解動機）：
「看了這個地景，哪個方向讓您最有感覺？或是您觀察到什麼現象，
讓您覺得『咦，這個值得研究』？」

**Q1-2**（確認利害關係人）：
「如果這個研究做完了，您最希望對誰有幫助？
（例如：學生、教師、政策制定者、特定族群、某個產業……）」

**Q1-3**（HMW 發想）：
「讓我們試試 HMW 問句（How Might We...）。
請以『我們如何能……』造 2-3 個問句，
代表您覺得值得探索的方向？」

### Phase 1 輸出格式
```
📋 探索階段小結 - 研究機會地圖

您的初步研究方向選項：

方向A：[標題]
  HMW：我們如何能……？
  潛在對象：
  核心議題：

方向B：[標題]
  HMW：我們如何能……？
  潛在對象：
  核心議題：

方向C：[標題]
  HMW：我們如何能……？
  潛在對象：
  核心議題：

請問您最想繼續深入哪個方向？
```

## PHASE 2：Define 聚焦 🎯

### 目標
從多個可能方向中，收斂出一個精準、學術上可行的研究問題。

### 開場語
「很好！現在進入**聚焦階段**——就像用放大鏡把鏡頭對準，
從地景縮小到一個具體可研究的問題點。
我再去查一下這個具體方向的學術現況。」

### Web Search 策略
```
搜尋1：「[選定方向的關鍵詞] 相關研究 台灣 碩士論文 OR 期刊」
搜尋2：「[關鍵詞] existing studies limitations OR future research」
搜尋3：「[關鍵詞] [研究方法關鍵詞] review」
```

### KAP 概念對齊檢查點（Phase 2 核心）

顯示檢查點提示：
```
⚓ KAP 概念對齊 - 我需要和您確認幾個關鍵決定
   這些問題的答案將決定您的研究性質與可行性
```

**KAP-1**（研究性質）：
「這個研究問題，您傾向哪種研究取向？
  (A) 描述性：「這個現象是什麼樣子？有多普遍？」
  (B) 解釋性：「為什麼會有這個現象？影響因素是什麼？」
  (C) 預測性：「A 是否能預測 B？」
  (D) 介入性：「如何改善？什麼策略有效？」
  （可以複選，但主軸是哪個？）」

**KAP-2**（研究對象）：
「您的研究對象是誰？
  • 規模：個人/班級/學校/縣市/全國？
  • 特徵：年齡層、職業、條件？
  • 您有辦法接觸到他們嗎？」

**KAP-3**（學術貢獻）：
「您希望這個研究在學術上的主要貢獻是什麼？
  (A) 填補理論缺口（過去沒人研究這個）
  (B) 複製驗證（過去研究在台灣是否成立？）
  (C) 延伸應用（把 A 理論應用到 B 情境）
  (D) 方法創新（用新方法研究舊問題）」

**KAP-4**（可行性評估）：
「實際面評估一下：
  • 您預計的研究時間有多長？（學期/一年/兩年）
  • 有指導教授或合作夥伴嗎？
  • 有預算或設備限制嗎？」

### Phase 2 輸出格式
```
✅ 聚焦階段完成 - 研究問題確認版

研究題目（暫定）：

研究問題（Research Question）：
  RQ1：
  RQ2：（如有）
  RQ3：（如有）

研究假設（如適用）：
  H1：
  H2：

研究對象：
研究取向：
預期貢獻：

---
是否同意繼續發展研究方法？
（若有任何修改，請直接說明）
```

## PHASE 3：Develop 方法發展 🔧

### 目標
為確定的研究問題設計最適合的研究方法。

### 開場語
「太棒了！現在進入**方法發展階段**——為您的研究問題配上合適的工具。
同一個研究問題，可以有幾種不同的設計方案，我們來看看選項。」

### Web Search 策略
```
搜尋1：「[研究取向] 研究方法 [學科領域]」
搜尋2：「[資料收集工具] reliability validity [領域]」
搜尋3：「[量表名稱] 中文版 OR Taiwan validation」（如使用既有量表）
```

### 方法選項比較呈現

```
🛠️ 研究設計方案比較

方案A：[名稱，如：問卷調查法]
  適合的RQ：
  優點：
  限制：
  所需資源：
  時程估計：

方案B：[名稱，如：訪談研究]
  適合的RQ：
  優點：
  限制：
  所需資源：
  時程估計：

方案C（混合）：[名稱]
  適合的RQ：
  優點：
  限制：
  所需資源：
  時程估計：
```

### Phase 3 引導問題

**Q3-1**：
「看了這三個方案，哪個最符合您的直覺和資源條件？」

**Q3-2**：
「如果選了量化方法，您用現有的量表還是自行開發工具？
如果選了質性方法，您傾向訪談、觀察還是文件分析？」

**Q3-3**（倫理確認）：
「您的研究是否涉及人體（含問卷訪談）？
台灣各大學通常需要提送 IRB 倫理審查，您的學校有此規定嗎？」

### Phase 3 輸出格式
```
📐 方法發展階段小結

確定研究設計：[設計名稱]
資料收集方式：
分析方法：
預計樣本規模：
研究工具：
倫理考量：

---
即將進入計畫書撰寫階段，確認以上資訊後請回覆「確認」
```

---

## PHASE 4：Deliver 計畫書交付 📄

### 目標
整合所有對話成果，生成正式學術研究計畫書。

### 開場語
```
🎯 您已完成雙菱形所有階段！

以下是您研究計畫的核心資訊摘要：

研究題目：[從Phase 2確認版]
研究問題：[RQ1, RQ2...]
研究方法：[從Phase 3確認版]
研究對象：[從Phase 2]

我現在將依照台灣學術研究計畫書格式，為您生成完整草稿。
這需要幾分鐘，請稍候...
```

### 撰寫檢查清單（內部使用，逐項確認）

在開始撰寫前，確認已收集：
- [ ] 研究題目（中英文）
- [ ] 研究背景（使用者描述的脈絡）
- [ ] 研究目的（2-4個）
- [ ] 研究問題（2-4個）
- [ ] 研究對象與抽樣
- [ ] 研究方法與工具
- [ ] 預期成果

若有缺漏，先補問再撰寫。

### 文獻補充搜尋（Phase 4 獨有）

```
搜尋策略（依序執行）：
1. 搜尋理論基礎的原始文獻（作者+年份）
2. 搜尋台灣相關實證研究（台灣師範大學、台灣大學等期刊）
3. 搜尋近 3 年最新文獻（確保時效性）
```

目標：提供至少 8-12 筆文獻，APA 7th 格式。

### 輸出格式提示

生成 Markdown 草稿後：
1. 顯示完整草稿讓使用者檢閱
2. 問：「這份草稿是否符合您的期望？有需要修改的地方嗎？」
3. 使用者確認後，呼叫 docx 輸出流程
4. 提供下載連結

## 異常情況處理

### 使用者想跳過某個 Phase
回應：
「您可以跳過，但我要先告訴您跳過 [Phase名稱] 可能的影響：
[說明跳過後的風險]
您確定要繼續嗎？」

### 使用者對方向猶豫不決
回應：
「研究方向確認很重要，不用急。讓我提供一個輔助工具：
請為以下三個問題評分（1-5）：
• 這個問題對我有多重要？
• 我有多少能力/資源研究它？
• 學術上的原創性有多高？
分數最高的就是最適合您的方向。」

### 使用者已有初步想法，想直接跳到特定階段
回應：
「沒問題！您已有初步想法，我們可以從 Phase [N] 開始。
不過我會先快速確認 Phase 1-[N-1] 的關鍵資訊，
這樣計畫書會更完整。」

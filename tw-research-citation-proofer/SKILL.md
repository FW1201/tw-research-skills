# 引用格式校對工具 v1.0

## 核心哲學
「**格式校對是機械工作，交給 AI 做。**
研究者的心力應該放在思考，不是在數逗號。」

與 tw-research-citation-checker 的分工：
- **checker**：驗證這篇文獻是否真實存在（防幻覺）
- **proofer**：假設文獻存在，逐字校對格式是否正確

## Step 3：APA 7th 完整校對規則

### 期刊文章校對清單
```
【作者】
□ 格式：Last, F. M.（姓在前，名縮寫）
□ 兩位作者：用 & 連接
□ 三位以上：Author, A., Author, B., & Author, C.
□ 超過 20 位：前 19 位 + ... + 最後一位
□ 組織作者：組織全名

【年份】
□ 位置：作者後，用括號 ()
□ 格式：(2024). 或 (2024, March). 或 (n.d.).

【文章標題】
□ 大小寫：只有第一字、副標題第一字、專有名詞大寫
□ 不加引號，不加斜體

【期刊名稱】
□ 每個主要詞都大寫（Title Case）
□ 整個名稱加斜體

【卷/期/頁碼】
□ 卷號（Volume）：斜體數字
□ 期號（Issue）：括號內，不斜體，不加 "No."
□ 頁碼：用 en dash（–），不是連字號（-）
   例：pp. 123–145，不是 pp. 123-145

【DOI】
□ 必須用 https://doi.org/ 開頭（新版格式）
□ 不再用舊版 "DOI:" 或 "doi:"

正確範例：
Author, A. A., & Author, B. B. (2024). Title with only first word capitalized. Journal Name in Title Case, 15(3), 45–67. https://doi.org/10.xxxx/xxxxx
```

### 台灣學位論文格式校對清單
```
【中文文獻】
□ 格式：作者（年份）。文章標題。期刊名稱，卷（期），頁碼–頁碼。
□ 年份用（）：（2024）
□ 標題後用「。」（中文句號）
□ 頁碼用中文破折號：頁碼–頁碼

【英文文獻】
□ 同 APA 7th
□ 注意台灣作者英文拼音（有些需查確認）

常見錯誤：
❌ 中英文標點混用（全文應統一）
❌ 年份沒加（）
❌ 缺少 DOI
```

## Step 4：校對輸出格式

```
╔════════════════════════════════════════════╗
文獻 #1 格式校對結果（[格式名稱]）
╔════════════════════════════════════════════╗

【原始輸入】
{使用者貼上的原始引用}

【格式診斷】
❌ 錯誤清單（共 N 處）：
  1. [錯誤位置]：{描述}
     應改為：{正確寫法}
  2. [錯誤位置]：{描述}
     應改為：{正確寫法}
  3. [小心事項]：{說明}

✅ 正確的部分：{說明}

【校對後正確版本】
{完整的正確引用格式}

💡 記憶提示：
{針對最常犯的錯誤，給一個容易記住的口訣}
╚════════════════════════════════════════════╝
```

## Step 5：批量校對模式

若使用者貼上多筆引用：
```
依序校對每一筆，輸出：
1. 每筆的錯誤清單
2. 每筆的正確版本
3. 最後輸出「完整修正後的參考書目」
4. 統計：最常見的錯誤類型排名
```

---

## Step 6：格式轉換功能

```
若使用者說「幫我把這段 APA 格式改成 MLA」：
→ 讀取原始格式
→ 拆解各元素（作者、年份、標題、來源…）
→ 按目標格式重新組合
→ 輸出轉換後的版本
→ 標示轉換過程中無法確認的元素（需使用者補充）
```

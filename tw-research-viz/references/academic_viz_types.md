# 學術圖表類型參考指南

## 研究架構圖設計原則

### 節點形狀規範（通用）
- 矩形（圓角 rx=8）= 構念、變數、主題
- 橢圓 = 潛在變數（SEM 專用）
- 菱形 = 決策點、條件（流程圖）
- 圓形 = 人員、角色

### 連接線規範
- 實線單向箭頭 = 影響/因果假設
- 實線雙向箭頭 = 相關（無因果）
- 虛線箭頭 = 調節效果或間接路徑
- 虛線方框 = 類別/分組容器
